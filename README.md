## Run Locally

1. Clone repo

   ```shell
    $ git clone https://github.com/PutuPutra/finance-portal.git
   ```

2. Install and Run

   ```shell
     npm install --legacy-peer-deps
     npm run dev
   ```

3. Auth

   Use the following credentials to log in:
